# ecommerce-backend

## Technologies

NodeJs
ExpressJs
MongoDB
Mongoose

## Installation

```bash
git clone https://github.com/vishwajeetgade/ecommerce-backend.git
```

```bash
cd ecommerce-backend
```

```bash
# install node_modules
npm install
```

```bash
# run on local environment
npm run dev
```

```bash
# run on production environment
npm start
```
